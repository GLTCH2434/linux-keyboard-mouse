# Customization

The default **linux-keyboard-mouse** layout is designed to provide a simple keyboard-based mouse experience.

The preset can be customized using **Input Remapper**.

---

## Changing Key Bindings

You can change any key mapping to match your preference.

Examples:

- Change movement keys from `WASD` to arrow keys.
- Change click keys.
- Change scrolling keys.
- Change the modifier key.

To modify the layout:

1. Open **Input Remapper**.
2. Select your keyboard.
3. Open the linux-keyboard-mouse preset.
4. Edit the key mappings.
5. Save the changes.
6. Apply the preset.

---

## Default Layout

The default configuration uses **Right Alt** as the modifier key.

### Movement

| Keys | Action |
|------|--------|
| Right Alt + W | Move up |
| Right Alt + A | Move left |
| Right Alt + S | Move down |
| Right Alt + D | Move right |

## Cursor Speed

The default preset provides two movement speeds:

- **Right Alt + WASD** for precise cursor movement.
- **Right Alt + M + WASD** for faster movement across the screen.

You can adjust the movement speed by editing the corresponding mouse movement actions in Input Remapper.

### Mouse Buttons

| Keys | Action |
|------|--------|
| Right Alt + K | Left click |
| Right Alt + L | Right click |

### Scrolling

| Keys | Action |
|------|--------|
| Right Alt + T | Scroll up |
| Right Alt + G | Scroll down |

---

## Saving Your Changes

After modifying the preset:

1. Save the preset in Input Remapper.
2. Test the new controls.
3. Keep a backup if you want to restore the original layout later.
